КЭД каскад во вращающемся электрическом поле

Рассмотрим модельную задачу о развитии КЭД каскада в однородном вращающемся (с частотой $\omega$) электрическом поле. Такая конфигурация реализуется, например, в узлах стоячей циркулярно-поляризованной волны

$$\mathbf{E}(t)=\{a_0\cos t, a_0\sin t\}$$

$$\dot{\mathbf{p}}(t)=\pm\mathbf{E}$$

Нормировка: \[\begin{align} && x \leftarrow \frac{\omega x}{c} && t \leftarrow \omega t && E \leftarrow \frac{eE}{mc\omega} && p \leftarrow \frac{p}{mc} && \end{align}\]

Решение (для позитрона): $$p_x(t)=p_{0x} + a_0 (\sin t - \sin t_0)$$ $$p_y(t)=p_{0y} - a_0 (\cos t - \cos t_0)$$

$$p_x(t)=p_{0x}+ a_0 (\sin t - \sin t_0)$$ $$p_y(t)=p_{0y}- a_0 (\cos t - \cos t_0)$$

При нулевых начальных условиях:

$$\gamma_e(t)=\sqrt{1+4a_0^2\sin^2\frac{ t}{2}}, \ \ \ \chi(t)=\frac{a_0}{a_S}\sqrt{1+4a_0^2\sin^4\frac{ t}{2}}$$ $$a_S = \frac{eE_{cr}}{mc\omega}\equiv\frac{mc^2}{\hbar\omega}$$

Будем рассматривать такие амплитуды полей $a_0$ при которых значение $\chi_e\sim 1$ достигается за время $t_{acc} \ll 1$, тогда

$$\gamma_e(t)\approx a_0 t$$ $$\chi_e(t)\approx \frac{a_0^2 t^2}{2 a_S}$$

$$\frac{1}{\sqrt{a_0}} \ll t \ll 1$$

$$\chi_e(t)\approx \frac{a_0^2 t^2}{2 a_S}$$

Определим время ускорения $t_{acc}$, при котором $\chi_e(t_{acc})=1$ $$t_{acc}\sim\frac{\sqrt{a_S}}{a_0}$$

Характерное "время жизни" электрона между излучениями: $$t_e \sim \frac{1}{W_{rad}}$$

В момент излучения:

\[\begin{align} && \gamma \approx \frac{a_0}{W_{rad}} && \chi \approx \frac{a_0^2}{2a_S W_{rad}^2} \end{align} \]

В момент излучения:

\[\begin{align} && \gamma \approx \frac{a_0}{W_{rad}} && \chi \approx \frac{a_0^2}{2a_S W_{rad}^2} \end{align} \]

Подставим в выражение для вероятности при $\chi \gg 1$:

$$W_{rad} \sim \frac{\alpha a_S}{\gamma} \chi^{2/3}$$

Получим:

\[\begin{align} && \gamma \approx \sqrt{a_S} {\left(\frac{a_0}{\alpha a_S} \right)}^{3/4} && t_e \approx \frac{1}{\alpha \sqrt{a_S}} {\left(\frac{a_0}{\alpha a_S} \right)}^{-1/4} \end{align} \]

Важно, что для развития каскада необходимо $t_e \gtrsim t_{acc}$, т.к. в таком случае электрон излучает фотон при $\chi \gtrsim 1$, при этом теряет почти половину своей энергии, поэтому для фотонов $\varepsilon_\gamma \sim \varepsilon_e$, $\chi_\gamma \sim \chi_e$. Условие $t_e \gtrsim t_{acc}$ выполняется при $a_0 /\alpha a_S \gtrsim 1$

При $a_0 /\alpha a_S \gtrsim 1$ и $\chi \gg 1$ формулы для вероятности излучения фотона электроном $W_{rad}$ и вероятности образования электрон-позитронной пары из фотона $W_{e^+e^-}$ отождествляются. В таком случае число частиц удваивается за время порядка $t_e$, т.е.

$$N_{e^\pm,\gamma}(t)\sim e^{\Gamma t}$$ $$\Gamma \sim \frac{1}{t_e}\sim \alpha \sqrt{a_S} {\left(\frac{a_0}{\alpha a_S} \right)}^{1/4} $$
